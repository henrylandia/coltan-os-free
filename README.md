# Coltan OS

A FreeBSD-based infrastructure platform for home and small/medium business environments.
Integrates firewall, file server, and backup system in a single web interface.

## Modules
- Networking & Firewall (PF, OpenVPN, WireGuard, DNS, DHCP)
- File Server (ZFS, Samba, NFS, iSCSI)
- Backup (Sanoid, Restic, remote destinations)
- Control Panel (dashboard, alerts, web console)

## Stack
- OS: FreeBSD 14.4
- Backend: Node.js + Fastify
- Frontend: HTML + Bootstrap
- Firewall: PF
- Storage: ZFS + Samba + NFS
- Backup: Sanoid + Restic
